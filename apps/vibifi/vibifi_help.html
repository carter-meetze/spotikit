Listifi




What is Listifi?

Listifi is a website that makes playlists based on certain criteria that you choose! After picking an artist you'd like to hear, you can choose some genres that you want to be included and customize the name of your playlist and the number of tracks included.


How does it work?

After filling out the form, this website accesses Spotify's API to create a playlist inside the app.
